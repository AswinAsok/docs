---
title: 'Export Guest Data'
description: 'Download guest information in XLSX format. Organizers can export guest data as a CSV file for record-keeping and analysis.'
---

### Steps to Download:

<Frame>
  <img src="/images/event/guests/export.png" className="rounded-lg shadow-md" />
</Frame>

1. Click the **CSV icon**  <Icon icon="file-csv" iconType="solid" color="grey" /> located at the top right of the Guests page.
2. A modal will open displaying available data fields.
3. Select the headers you want to include in the exported file.
4. Click **Download** to generate and save the CSV file.
